Javascript Language

20th August 2017

In the JavaScript language, describe all the use cases for each of: parentheses () - e.g. '1. Execute a function var result = doSomething()'

Parenthesis () in JavaScript are used for function calls, to surround conditional statements, or for grouping to enforce Order of Operations.

Parentheses have multiple functions relating to functions and structures. They are used to contain a list of parameters passed to functions and control structures and they are used to group expressions to control the order of execution. Some functions have no parameters and in this case, the space between parentheses is blank.

In the JavaScript language, describe all the use cases for brackets []

Square brackets are used in objects and when defining variables. They can also be used when referring to the position of an item in an array.

In the JavaScript language, describe all the use cases for braces {}

A pair of curly braces {...} delimits the body code when declaring a function. They also used for grouping statements together into a single statement that creates a block of code to be executed.

In the JavaScript language, describe all the use cases for single quotes '' or double quotes" "

The most likely reason for use of single vs double in different libraries is programmer preference and/or API consistency. Other than being consistent, use whichever best suits the string. I prefer single quotes as you don't have to push shift to get them.
